# Учебная тетрадь: Фронтенд и бэкенд разработка

## 📚 Описание

Данный репозиторий представляет собой мою учебную тетрадь по предмету **«Фронтенд и бэкенд разработка»**.  
Здесь собраны практические задания и примеры реализации веб-приложений.

Цель тетради — закрепление знаний по разработке современных веб-приложений и практическое применение изученных технологий.

---

## 🎯 Цели курса

- Изучение основ разработки бэкенда
- Создание пользовательских интерфейсов (фронтенд)
- Взаимодействие фронтенда и бэкенда через API
- Реализация авторизации и аутентификации
- Использование современных инструментов разработки

---

## ⚙️ Используемые технологии

### Бэкенд
- Node.js / Express.js   
- REST API   

### Фронтенд
- HTML  
- CSS  
- JavaScript  
- React / Next.js  

### Инструменты
- Git / GitHub  
- VS Code  
- Postman
