    📝 О приложении

    Версия 1.2.3

    • 📝 Прогрессивное веб-приложение (PWA) для управления задачами.
    • ⚡ Работает офлайн благодаря Service Worker и Cache API.
    • 🔄 Синхронизация в реальном времени через WebSocket (Socket.IO).
    • 🔔 Push-уведомления о новых задачах даже при закрытой вкладке.
    • 💾 Данные сохраняются в localStorage и доступны без сети.
    • ⏰ Планирование напоминаний с кнопкой «Отложить на 5 минут».

    🛠 Технологии

    HTML / CSS Service Worker Web App Manifest Socket.IO Web Push API localStorage
