📝 О приложении

Версия 1.2.3

  • 📝 Заметки — это прогрессивное веб-приложение (PWA) для управления задачами.
  • ⚡ Работает офлайн благодаря Service Worker и Cache API.
  • 🏗 Использует архитектуру App Shell — каркас загружается мгновенно из кэша.
  • 💾 Данные сохраняются в localStorage и остаются доступны даже без сети.
  • 🔒 Работает по HTTPS с локальным доверенным сертификатом (mkcert).

🛠 Технологии

HTML / CSS Service Worker Web App Manifest localStorage HTTPS
